Cmajor 4.1.0 for Windows x64 and Linux

Release notes

  • Code completion improved.

  • Installation document revised.

  • Using new installer and uninstaller programs on Windows.

  • The new installer and uninstaller require that Visual C++ Runtime Libraries are installed to the computer to which the Cmajor is deployed.
    The Visual C++ Runtime Libraries can be downloaded here:
    https://support.microsoft.com/en-us/topic/the-latest-supported-visual-c-downloads-2647da03-1eea-4433-9aff-95f26a218cc0
    Download and run the vc_redist.x64.exe package.
