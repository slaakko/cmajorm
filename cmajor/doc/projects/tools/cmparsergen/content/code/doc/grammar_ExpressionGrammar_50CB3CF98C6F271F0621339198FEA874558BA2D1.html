ExpressionGrammar Grammar

Expression → Disjunction
Disjunction → (Conjunction ("||" Conjunction)*)
Conjunction → (BitOr ("&&" BitOr)*)
BitOr → (BitXor (('|' − "||") BitXor)*)
BitXor → (BitAnd ('^' BitAnd)*)
BitAnd → (Equality (('&' − "&&") Equality)*)
Equality → (Relational (("==" | "!=") Relational!)*)
Relational → (Shift (("<=" | ">=" | ('<' − "<<") | ('>' − ">>") | (keyword("is") TypeExpr) | (keyword("as") TypeExpr)) (Shift | empty))*)
Shift → (Additive (("<<" | ">>") empty Additive!)*)
Additive → (Multiplicative ((('+' − "++") | ('-' − "--")) Multiplicative!)*)
Multiplicative → (Prefix (('*' | '/' | '%') Prefix!)*)
Prefix → (("++" | "--" | '-' | '+' | ('!' − "!=") | '~' | '*' | ('&' − "&&")) Prefix) | Postfix
Postfix → (Primary ("++" | "--" | '.' Identifier | "->" Identifier | ('[' Expression ']'!) | ('(' ArgumentList ')'!))*)
Primary → ('(' Expression ')') | Literal | BasicType | TemplateId | Identifier | keyword("this") | keyword("base") | SizeOfExpr | TypeNameExpr | CastExpr | ConstructExpr | NewExpr
SizeOfExpr → (keyword("sizeof") '('! Expression! ')'!)
TypeNameExpr → (keyword("typename") '('! Expression! ')'!)
CastExpr → (keyword("cast") '<'! TypeExpr! '>'! '('! Expression! ')'!)
ConstructExpr → keyword("construct") '<'! TypeExpr! '>'! '('! ExpressionList! ')'!
NewExpr → keyword("new") TypeExpr! ('(' ArgumentList ')')?
ArgumentList → ExpressionList?
ExpressionList → empty ((Expression) % ',')
