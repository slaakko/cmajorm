PrimaryGrammar Grammar

Primary → (RuleCall | Primitive | Grouping | Token) (Expectation)? (Action)?
RuleCall → ((token(Nonterminal '(') ArgumentList ')' | Nonterminal) Alias?)
Nonterminal → Identifier
Alias → ':' Identifier
Grouping → ('(' Alternative! ')'!)
Token → (keyword("token") '('! Alternative! ')'!)
Expectation → '!'
Action → (CompoundStatement ('/' CompoundStatement)?)
