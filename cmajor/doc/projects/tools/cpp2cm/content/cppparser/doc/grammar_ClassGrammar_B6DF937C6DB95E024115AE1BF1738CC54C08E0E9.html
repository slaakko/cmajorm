ClassGrammar Grammar

ClassDeclaration → (ClassSpecifier ';')
ForwardClassDeclaration → (ClassKey ClassName ';')
ClassSpecifier → ClassHead '{' MemberSpecifications '}'!
MemberSpecifications → MemberSpecification*
MemberSpecification → AccessSpecifier ':' (MemberDeclaration)* | (MemberDeclaration)+
MemberDeclaration → (DeclSpecifiers TypeExpr Declarator VirtPureSpecifiers BraceOrEqualInitializer? ';') | SpecialMemberFunctionDeclaration | (FunctionDefinition ';'?) | SpecialMemberFunctionDefinition | UsingDeclaration | TypedefDeclaration | TemplateDeclaration | ClassDeclaration | EnumDeclaration | AliasDeclaration | ForwardClassDeclaration
SpecialMemberFunctionDeclaration → (DeclSpecifiers Declarator VirtPureSpecifiers ';')
ClassHead → (ClassKey ClassName ClassVirtSpecifiers BaseClause?)
ClassKey → keyword("class") | keyword("struct") | keyword("union")
ClassName → QualifiedId
ClassVirtSpecifiers → (ClassVirtSpecifier)*
ClassVirtSpecifier → keyword("final") | keyword("explicit")
BaseClause → ':' BaseClassSpecifierList
BaseClassSpecifierList → BaseClassSpecifier (',' BaseClassSpecifier)*
BaseClassSpecifier → (BaseSpecifiers ClassName)
BaseSpecifiers → (BaseSpecifier)*
BaseSpecifier → keyword("virtual") | AccessSpecifier
AccessSpecifier → keyword("public") | keyword("protected") | keyword("private")
VirtPureSpecifiers → (VirtSpecifier)* (PureSpecifier)?
VirtSpecifier → keyword("override") | keyword("final") | keyword("new")
PureSpecifier → ('=' '0')
SpecialMemberFunctionDefinition → (DeclSpecifiers Declarator (CtorInitializer)? (FunctionBody | '=' (keyword("default") | keyword("delete")) ';'))
CtorInitializer → ':' MemberInitializerList
MemberInitializerList → MemberInitializer (',' MemberInitializer)*
MemberInitializer → (MemberInitializerId '(' PossiblyEmptyArgumentList ')') | (MemberInitializerId BracedInitializerList)
MemberInitializerId → QualifiedId
