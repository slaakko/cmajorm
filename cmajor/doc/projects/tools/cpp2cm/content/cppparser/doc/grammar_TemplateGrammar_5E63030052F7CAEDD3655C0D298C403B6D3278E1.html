TemplateGrammar Grammar

TemplateDeclaration → keyword("template") '<' TemplateParameterList! '>'! (Declaration)!
TemplateParameterList → TemplateParameter (',' TemplateParameter)*
TemplateParameter → TypeParameter | ParameterDeclaration
TypeParameter → ((keyword("class") | keyword("typename")) Identifier ('=' TypeExpr)?)
SimpleTemplateId → (TemplateName '<' TemplateArgumentList? '>')
TemplateId → SimpleTemplateId
TemplateName → Identifier
TemplateArgumentList → TemplateArgument (',' TemplateArgument)*
TemplateArgument → TypeExpr | ConditionalExpression
ExplicitInstantiation → (keyword("template") Declaration)
ExplicitSpecialization → (keyword("template") '<' '>' Declaration)
