DeclaratorGrammar Grammar

InitDeclarator → Declarator (Initializer | empty)
Declarator → ((IdExpression | ('(' '*' Identifier ')') | ('(' NestedNameSpecifier '*' Identifier ')')) (empty ParameterList CVSpecifierSequence | ('[' ConstantExpression? ']'!))*)
Initializer → ('=' InitializerClause) | '(' ExpressionList ')' | BracedInitializerList
BraceOrEqualInitializer → ('=' InitializerClause) | BracedInitializerList
BracedInitializerList → ('{' (InitializerClause (',' InitializerClause)*)? '}')
InitializerClause → AssignmentExpression | BracedInitializerList
InitializerList → empty (InitializerClause (',' InitializerClause)*)
