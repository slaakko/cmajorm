EnumerationGrammar Grammar

EnumDeclaration → (EnumSpecifier ';')
OpaqueEnumDeclaration → (EnumKey EnumName EnumBase? ';')
EnumSpecifier → EnumHead '{' Enumerators '}'!
EnumHead → (EnumKey EnumName EnumBase?)
EnumKey → (keyword("enum") keyword("class")) | (keyword("enum") keyword("struct")) | keyword("enum")
EnumName → QualifiedId
EnumBase → (':' TypeExpr)
Enumerators → EnumeratorDefinition (',' EnumeratorDefinition)*
EnumeratorDefinition → (Enumerator ('=' ConstantExpression)?)
Enumerator → Identifier
