IntermediateCodeFileGrammar Grammar

IntermediateCodeFile → empty CompileUnitHeader TypeDeclarations? DataDefinitions? Functions? Metadata?
CompileUnitHeader → (keyword("cu") '(' (hexdigit+) ',' MDStructRef ')')
TypeDeclarations → (keyword("types") '{'! TypeDeclaration* '}'!)
DataDefinitions → (keyword("data") '{'! DataDefinition* '}'!)
Functions → Function*
Function → FunctionHeader '{'! BasicBlock* '}'!
FunctionHeader → keyword("function") TypeExpr! (keyword("once"))? identifier! (MDStructRef?)
BasicBlock → token(Label S InstructionLine+ EmptyLine?)
EmptyLine → token(S? newline)
InstructionLine → token(S? Instruction (S MDStructRef)? S? newline!)
Instruction → StoreInstruction | ArgInstruction | JumpInstruction | BranchInstruction | ProcedureCallInstruction | RetInstruction | SwitchInstruction | ValueInstruction | NoOperationInstruction | SaveInstruction
StoreInstruction → (keyword("store") S TypeExpr! S Value! Comma! TypeExpr! S Value!)
ArgInstruction → (keyword("arg") S TypeExpr! S Value!)
JumpInstruction → (keyword("jmp") S Label!)
BranchInstruction → (keyword("branch") S TypeExpr! S Value! Comma! Label! Comma! Label!)
ProcedureCallInstruction → (keyword("call") S TypeExpr! S Value!)
RetInstruction → keyword("ret") S ((keyword("void") − "void*") | TypeExpr S Value)
SwitchInstruction → (keyword("switch") S TypeExpr! S Value! S Label!) Comma! '['! S? (TypeExpr S Value Comma Label) % (S? ':' S?) ']'!
NoOperationInstruction → keyword("nop")
SaveInstruction → keyword("save")
ValueInstruction → (TypeExpr S Value! S? '=' S? Operation!)
Operation → UnaryInstruction | BinaryInstruction | ParamInstruction | LocalInstruction | LoadInstruction | ElemAddrInstruction | PtrOffsetInstruction | PtrDiffInstruction | FunctionCallInstruction | TrapInstruction
UnaryInstruction → NotInstruction | NegInstruction | SignExtendInstruction | ZeroExtendInstruction | TruncateInstruction | BitCastInstruction | IntToFloatInstruction | FloatToIntInstruction | IntToPtrInstruction | PtrToIntInstruction
NotInstruction → (keyword("not") S Operand!)
NegInstruction → (keyword("neg") S Operand!)
SignExtendInstruction → (keyword("signextend") S Operand!)
ZeroExtendInstruction → (keyword("zeroextend") S Operand!)
TruncateInstruction → (keyword("truncate") S Operand!)
BitCastInstruction → (keyword("bitcast") S Operand!)
IntToFloatInstruction → (keyword("inttofloat") S Operand!)
FloatToIntInstruction → (keyword("floattoint") S Operand!)
IntToPtrInstruction → (keyword("inttoptr") S Operand!)
PtrToIntInstruction → (keyword("ptrtoint") S Operand!)
BinaryInstruction → AddInstruction | SubInstruction | MulInstruction | DivInstruction | ModInstruction | AndInstruction | OrInstruction | XorInstruction | ShlInstruction | ShrInstruction | EqualInstruction | LessInstruction
AddInstruction → (keyword("add") S Operand! Comma! Operand!)
SubInstruction → (keyword("sub") S Operand! Comma! Operand!)
MulInstruction → (keyword("mul") S Operand! Comma! Operand!)
DivInstruction → (keyword("div") S Operand! Comma! Operand!)
ModInstruction → (keyword("mod") S Operand! Comma! Operand!)
AndInstruction → (keyword("and") S Operand! Comma! Operand!)
OrInstruction → (keyword("or") S Operand! Comma! Operand!)
XorInstruction → (keyword("xor") S Operand! Comma! Operand!)
ShlInstruction → (keyword("shl") S Operand! Comma! Operand!)
ShrInstruction → (keyword("shr") S Operand! Comma! Operand!)
EqualInstruction → (keyword("equal") S Operand! Comma! Operand!)
LessInstruction → (keyword("less") S Operand! Comma! Operand!)
ParamInstruction → keyword("param")
LocalInstruction → (keyword("local") S TypeExpr!)
LoadInstruction → (keyword("load") S Operand!)
ElemAddrInstruction → (keyword("elemaddr") S Operand! Comma! Operand!)
PtrOffsetInstruction → (keyword("ptroffset") S Operand! Comma! Operand!)
PtrDiffInstruction → (keyword("ptrdiff") S Operand! Comma! Operand!)
FunctionCallInstruction → (keyword("call") S TypeExpr! S Value!)
TrapInstruction → (keyword("trap") S TypeExpr! S Value! Comma! TypeExpr! S Value! Comma! TypeExpr! S Value!)
Operand → (TypeExpr S Value!)
Label → token('@' uint)
Value → IdValue | SymbolValue | LiteralValue
IdValue → ('$' uint)
SymbolValue → token('@' identifier)
LiteralValue → (token(anychar − (',' | S | ')' | newline))+)
S → [ \t]+
Comma → S? ',' S?
VariableName → token((idstart | '_' | '@') (idcont | '@')*)
TypeDeclaration → StrTypeId '='! keyword("type")! (StructureType | ArrayType | FunctionType)
StructureType → ('{' ((StrTypeId! (':' "offset" '=' long)?) % ',')? '}'!) (keyword("size") '=' long ',' keyword("alignment") '=' long)?
ArrayType → ('[' long! 'x'! StrTypeId! ']'!) (keyword("size") '=' long ',' keyword("alignment") '=' long)?
FunctionType → (keyword("function") StrTypeId! '('! (StrTypeId % ',')? ')'!)
StrTypeId → (token("$T" uint) | (keyword_list(identifier, ["void", "byte", "float", "uint", "bool", "long", "ushort", "ulong", "double", "int", "sbyte", "short"]))) (('*'))*
TypeExpr → PostfixTypeExpr
PostfixTypeExpr → PrimaryTypeExpr ('*')*
PrimaryTypeExpr → TypeId | PrimitiveType
TypeId → token("$T" uint)
PrimitiveType → VoidType | BoolType | SByteType | ByteType | ShortType | UShortType | IntType | UIntType | LongType | ULongType | FloatType | DoubleType
VoidType → keyword("void")
BoolType → keyword("bool")
SByteType → keyword("sbyte")
ByteType → keyword("byte")
ShortType → keyword("short")
UShortType → keyword("ushort")
IntType → keyword("int")
UIntType → keyword("uint")
LongType → keyword("long")
ULongType → keyword("ulong")
FloatType → keyword("float")
DoubleType → keyword("double")
DataDefinition → TypeExpr (keyword("once"))? VariableName! (';' | '='! Constant)
Constant → ConversionConstant | ClsIdConstant | PtrConstant | SymbolConstant | BoolConstant | SByteConstant | ByteConstant | ShortConstant | UShortConstant | IntConstant | UIntConstant | LongConstant | ULongConstant | FloatConstant | DoubleConstant | ArrayConstant | StructureConstant | StringConstant | StringArrayConstant
BoolConstant → (BoolType BoolValue)
BoolValue → bool
SByteConstant → SByteType SByteValue
SByteValue → sbyte
ByteConstant → ByteType ByteValue
ByteValue → byte
ShortConstant → ShortType ShortValue
ShortValue → short
UShortConstant → UShortType UShortValue
UShortValue → ushort
IntConstant → IntType IntValue
IntValue → int
UIntConstant → UIntType UIntValue
UIntValue → uint
LongConstant → LongType LongValue
LongValue → long
ULongConstant → ULongType ULongValue
ULongValue → ulong
FloatConstant → FloatType FloatValue
FloatValue → float
DoubleConstant → DoubleType DoubleValue
DoubleValue → double
ConversionConstant → (TypeExpr keyword("conv") '('! Constant! ')'!)
ClsIdConstant → TypeExpr keyword("clsid") '('! (hexdigit+) ')'!
PtrConstant → (TypeExpr keyword("null")) | (TypeExpr LiteralValue)
SymbolConstant → TypeExpr SymbolValue
ArrayConstant → '[' (Constant % ',')? ']'
StructureConstant → '{' (Constant % ',')? '}'
StringConstant → token('\"' ([^\"]*) '\"')
StringArrayConstant → StringArrayPrefix '[' (Constant % ',')? ']'
StringArrayPrefix → 'w' | 'u' | 'b'
Metadata → keyword("metadata") '{'! MDStruct* '}'!
MDItem → MDBool | MDLong | MDString | MDStructRef
MDBool → keyword("true") | keyword("false")
MDLong → long
MDString → string
MDStructRef → ('!' int)
MDStruct → '!' int '=' '{' (MDField % ',')? '}'
MDField → (identifier ':' MDItem)
