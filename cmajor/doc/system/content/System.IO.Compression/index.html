System.IO.Compression Module

The System.IO.Compression module contains data compression/decompression streams. It is implemented by means of zlib and bzip2 libraries.

Examples

This deflate program compresses a file using DeflateStream or extracts a file compressed using DeflateStream .

This bz2 program compresses a file using BZip2Stream or extracts a file compressed using BZip2Stream .
