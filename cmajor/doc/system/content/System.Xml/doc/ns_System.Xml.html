ParseXmlContent Function

public void ParseXmlContent(const String<char>& xmlContent, const String<char>& systemId, XmlContentHandler* contentHandler)

Parses an UTF-8 encoded XML string with a content handler derived from the XmlContentHandler class. The base class content handler does nothing by default.

The systemId parameter is used for error reporting only and will occur in the possible error message. If the XML string is not syntactically valid, the function throws an ExpectationFailure exception. If the XML string is not otherwise well-formed, the function throws an XmlProcessingException exception.
