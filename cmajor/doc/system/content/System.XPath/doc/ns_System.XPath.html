Evaluate Function

public UniquePtr<XPathObject> Evaluate(const String<uchar>& xpathExpression, DomDocument* document)

Compiles the given XPath query expression , evaluates it with respect to the given DomDocument and returns the resulting XPathObject .

The concrete type of the result depends on the supplied XPath expression. If the expression is a path expression the result will be a XPathNodeSet . If the expression is a Boolean expression, for example an or expression , the result will be a XPathBoolean , and so on. The result of a function call has a type depending on the function called. The functions implemented are documented under the XPathFunctionLibrary .

If the query string is not syntactically valid XPath expression , the function throws an ExpectationFailure exception. If an error such as type violation is detected when evaluating the query expression, the function throws an XPathException exception.
