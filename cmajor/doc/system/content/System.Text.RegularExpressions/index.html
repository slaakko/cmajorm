System.Text.RegularExpressions Module

The System.Text.RegularExpressions contains a minimal implementation of a regular expression matching engine.

Example

This rex program searches a regular expression pattern from the given text files. A pattern can be compiled to a DFA or represented as an NFA . Names of files can contain wildcards * and ?.
