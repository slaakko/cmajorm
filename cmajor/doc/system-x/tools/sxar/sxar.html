System X Archiver

The System X Archiver bundles System X object files (*.o) to an archive FILE.a.

Usage: sxar [options] { FILE.o }

long option short option description
‑‑help ‑h Print help and exit.
‑‑verbose ‑v Be verbose.
‑‑archive ARCHIVE.a ‑a ARCHIVE.a Set generated archive file name to ARCHIVE.a.
