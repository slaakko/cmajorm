Json Grammar

Rules

Value → String | Number | Object | Array | keyword("true") | keyword("false") | keyword("null")
String → token('\"' ([^\"\\] | ("\\u" (hexdigit hexdigit hexdigit hexdigit)) | escape)* '\"')
Number → num
Object → '{' ((String ':' Value) % ',')? '}'
Array → '[' (Value % ',')? ']'
