Project Grammar

Rules

Project → keyword("project") qualified_id ';' Declaration*
Declaration → ReferenceDeclaration | SourceFileDeclaration | TextFileDeclaration | TargetDeclaration
ReferenceDeclaration → keyword("reference") FilePath! ';'!
SourceFileDeclaration → keyword("source") FilePath! ';'!
TextFileDeclaration → keyword("text") FilePath! ';'!
TargetDeclaration → keyword("target") '='! Target! ';'!
Target → keyword("program") | keyword("library") | keyword("unitTest")
FilePath → token('<' ([^>]+) '>'!)
