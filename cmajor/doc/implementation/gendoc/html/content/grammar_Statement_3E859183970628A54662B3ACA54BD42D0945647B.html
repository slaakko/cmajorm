Statement Grammar

Rules

Statement → LabeledStatement | ControlStatement | ExpressionStatement | AssignmentStatement | ConstructionStatement | DeleteStatement | DestroyStatement | EmptyStatement | ThrowStatement | TryStatement | AssertStatement | ConditionalCompilationStatement
LabelId → identifier − Keyword
Label → LabelId ':'
LabeledStatement → Label Statement
ControlStatement → CompoundStatement | ReturnStatement | IfStatement | WhileStatement | DoStatement | RangeForStatement | ForStatement | BreakStatement | ContinueStatement | GotoStatement | SwitchStatement | GotoCaseStatement | GotoDefaultStatement
CompoundStatement → '{' Statement* '}'!
ReturnStatement → keyword("return") Expression? ';'!
IfStatement → keyword("if") '(' Expression ')' Statement (keyword("else") Statement)?
WhileStatement → keyword("while") '('! Expression! ')'! Statement!
DoStatement → keyword("do") Statement! keyword("while")! '('! Expression! ')'! ';'!
ForStatement → keyword("for") '(' ForInitStatement! Expression? ';'! ForLoopStatementExpr! ')'! Statement!
ForInitStatement → AssignmentStatement | ConstructionStatement | EmptyStatement
ForLoopStatementExpr → AssignmentStatementExpr | Expression | empty
RangeForStatement → keyword("for") '(' TypeExpr Identifier ':' Expression! ')'! Statement!
BreakStatement → keyword("break") ';'!
ContinueStatement → keyword("continue") ';'!
GotoStatement → keyword("goto") Identifier ';'!
SwitchStatement → (keyword("switch") '('! Expression! ')'!) '{'! (CaseStatement | DefaultStatement)* '}'!
CaseStatement → ((keyword("case") Expression ':'!)+) Statement*)
DefaultStatement → (keyword("default") ':'!) Statement*
GotoCaseStatement → keyword("goto") keyword("case") Expression! ';'!
GotoDefaultStatement → keyword("goto") keyword("default") ';'!
AssignmentStatementExpr → (Expression '=' Expression)
AssignmentStatement → AssignmentStatementExpr ';'
ConstructionStatement → (TypeExpr Identifier!) ('=' Expression! | '(' ArgumentList! ')'! | empty) ';'!
DeleteStatement → keyword("delete") Expression! ';'!
DestroyStatement → keyword("destroy") Expression! ';'!
ExpressionStatement → (Expression ';')
EmptyStatement → ';'
ThrowStatement → keyword("throw") Expression? ';'!
TryStatement → (keyword("try") CompoundStatement) Catch+
Catch → keyword("catch") '('! TypeExpr! Identifier? ')'! CompoundStatement
AssertStatement → '#' keyword("assert") Expression! ';'!
ConditionalCompilationStatement → ('#' keyword("if") '('! ConditionalCompilationExpression ')'! Statement* ('#' keyword("elif") '('! ConditionalCompilationExpression ')'! Statement*)* ('#' keyword("else") Statement* '#' keyword("endif")!)
ConditionalCompilationExpression → ConditionalCompilationDisjunction
ConditionalCompilationDisjunction → ConditionalCompilationConjunction ("||" ConditionalCompilationConjunction)*
ConditionalCompilationConjunction → ConditionalCompilationPrefix ("&&" ConditionalCompilationPrefix)*
ConditionalCompilationPrefix → ('!' ConditionalCompilationPrefix!) | ConditionalCompilationPrimary!
ConditionalCompilationPrimary → Symbol | '(' ConditionalCompilationExpression ')'
Symbol → identifier − Keyword
