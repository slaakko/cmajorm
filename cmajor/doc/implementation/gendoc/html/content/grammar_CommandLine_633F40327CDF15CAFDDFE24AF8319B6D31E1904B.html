CommandLine Grammar

Rules

CommandLine → Spaces? Argument (Spaces Argument)* Spaces?
Spaces → [\t ]+
Argument → ArgElement+
ArgElement → OddBackslashesAndLiteralQuotationMark | EvenBackslashesAndQuotationMark StringChar* EvenBackslashesAndQuotationMark | [^\t ]
OddBackslashesAndLiteralQuotationMark → "\\"+ '\"'
EvenBackslashesAndQuotationMark → "\\"* '\"'
StringChar → OddBackslashesAndLiteralQuotationMark | (anychar − EvenBackslashesAndQuotationMark)
